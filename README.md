# Village Map AI → Naksha AI (Mobile)

The original web UI lives in [`village-map-ai-main/`](village-map-ai-main/) (TanStack Start).

The **Expo React Native** app **Naksha AI** is in [`naksha-ai-temp/`](naksha-ai-temp/).

## Quick start (Android + Expo Go)

```bash
cd naksha-ai-temp
npm install
npm start
```

Scan the QR code with Expo Go. See [naksha-ai-temp/README.md](naksha-ai-temp/README.md) for details.
